# Notice Domain

## 역할

`notice` 도메인은 공지 작성, 대상 설정, 이미지/링크/투표 콘텐츠, 알림 발송과 읽음 상태를 관리한다.

## 책임

- 공지를 작성, 수정, 발행, 조회한다.
- 기수, 지부, 학교, 역할 기준으로 공지 대상을 정한다.
- 공지 이미지, 링크, 투표를 관리한다.
- 공지 알림과 리마인드를 발송한다.

## 경계

공지 대상의 실제 회원·챌린저·조직 정보는 각 도메인에서 관리한다. notice는 대상 조건을 보관하고 조회/발송 시 필요한 데이터만 공개 UseCase로 가져온다.

## UX Writing Notes

공지 문구는 운영자가 빠르게 수정할 수 있어야 한다. `공지 수신자 설정이 올바르지 않아요. 대상 설정을 확인해주세요`처럼 확인할 위치를 알려준다.
