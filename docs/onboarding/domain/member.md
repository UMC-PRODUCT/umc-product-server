# Member Domain

## 역할

`member` 도메인은 회원 계정, 프로필, 학교 소속, 이메일 기반 자격증명과 회원 검색을 관리한다.

## 책임

- 회원가입과 회원 상태를 관리한다.
- 프로필 이미지, 링크, 닉네임 등 공개 프로필을 관리한다.
- 로그인 ID와 비밀번호 자격증명 등록 여부를 관리한다.
- 회원 검색과 요약 조회를 제공한다.

## 경계

회원은 사용자 정체성의 중심이지만, 챌린저 활동 기록은 `challenger`, 조직 배정은 `organization`, 약관 동의는 `term` 도메인의 책임이다. 다른 도메인 정보는 ID와 공개 UseCase로 연결한다.

## UX Writing Notes

회원 오류는 사용자가 직접 해결하는 경우가 많다. `로그인해주세요`, `다른 이메일을 입력해주세요`, `학교 정보를 먼저 등록해주세요`처럼 행동을 분명히 쓴다.
