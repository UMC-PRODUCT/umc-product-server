# Schedule Domain

## 역할

`schedule` 도메인은 UMC 활동 일정, 참여자, 출석 정책, 출석 요청과 승인/거절을 관리한다.

## 책임

- 일정을 생성, 수정, 삭제한다.
- 대면/비대면, 위치, 태그, 참여자를 검증한다.
- 출석 정책과 출석 가능 시간을 관리한다.
- 출석 요청, 사유 제출, 운영진 승인/거절을 처리한다.

## 경계

일정 생성 자격은 챌린저 활동 기록과 조직 권한에 의존한다. schedule은 필요한 ID와 권한 컨텍스트만 받아 출석 규칙을 판단한다.

## UX Writing Notes

출석 오류는 사용자가 즉시 행동할 수 있어야 한다. `출석 가능 시간 이후에 다시 시도해주세요`, `참여자 목록을 확인해주세요`, `출석 정책을 먼저 설정해주세요`처럼 다음 행동을 구체화한다.
