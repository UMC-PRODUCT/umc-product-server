# Organization Domain

## 역할

`organization` 도메인은 기수, 지부, 학교, 스터디 그룹, 스터디 그룹 일정 연결을 관리한다. UMC 운영 구조의 기준 데이터를 제공한다.

## 책임

- 기수를 생성, 수정, 삭제하고 활성 기수를 조회한다.
- 지부와 학교를 생성하고 배정한다.
- 스터디 그룹, 멤버, 파트장을 관리한다.
- 조직 범위별 권한 판단에 필요한 컨텍스트를 제공한다.

## 경계

organization은 조직 구조를 소유하지만, 회원 상세 정보와 챌린저 활동 정보는 각각 `member`, `challenger` 도메인에서 관리한다. 스터디 그룹 구성원은 ID로 연결한다.

## UX Writing Notes

입력 오류는 `선택해주세요`, `입력해주세요`로 바로 안내한다. 삭제나 배정 실패는 `연결 정보를 먼저 정리해주세요`, `학교 배정 정보를 확인해주세요`처럼 운영자가 확인할 데이터를 말한다.
