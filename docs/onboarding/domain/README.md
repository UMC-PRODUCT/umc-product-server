# Domain Onboarding

이 디렉터리는 UMC PRODUCT의 제품 도메인별 역할, 책임, 경계, UX 문구 기준을 정리한다.

## 문서 범위

문서는 `domain/application/adapter` 구조를 갖고 실제 제품 기능과 사용자 오류 표면을 가진 도메인을 기준으로 작성했다. `common`, `global`, `test`는 여러 도메인이 공유하는 지원 모듈이므로 별도 도메인 문서로 분리하지 않았다.

## 작성 원칙

도메인 문서는 업무 규칙을 장황하게 반복하지 않는다. 새로 합류한 사람이 “이 도메인이 무엇을 소유하고, 어떤 말로 사용자에게 설명해야 하는지”를 빠르게 파악할 수 있는 수준으로 유지한다.
