# LLM Domain

## 역할

`llm` 도메인은 AI 응답 생성을 위한 채팅 완료 호출, 제공자 설정, 호출 제한과 회로 차단을 관리한다.

## 책임

- AI 제공자 설정을 확인한다.
- 채팅 완료 요청을 외부 LLM 제공자에게 전달한다.
- 응답 형식을 검증한다.
- 호출 제한과 장애 상태를 관리한다.

## 경계

LLM 도메인은 응답 생성 자체를 담당하지만, 어떤 업무에서 AI를 사용할지는 호출한 도메인이 결정한다. 외부 제공자의 장애는 사용자에게 과도한 기술 정보 없이 재시도 안내로 전달한다.

## UX Writing Notes

사용자에게는 `LLM`보다 `AI 응답`이 더 쉽다. `AI 응답을 생성하지 못했어요. 잠시 후 다시 시도해주세요`처럼 짧게 쓴다.
