# Storage Domain

## 역할

`storage` 도메인은 파일 업로드 준비, 업로드 완료 확인, 파일 접근 링크 생성, 삭제를 관리한다.

## 책임

- 파일 메타데이터를 저장하고 조회한다.
- 업로드 가능한 파일 형식과 크기를 검증한다.
- S3 기반 저장소 어댑터를 통해 업로드/삭제/접근 링크를 처리한다.
- CDN 접근 링크 설정을 관리한다.

## 경계

storage는 파일 자체와 접근 링크를 관리한다. 파일이 어떤 도메인의 첨부물인지, 삭제 가능한 업무 상태인지는 호출한 도메인이 판단한다.

## UX Writing Notes

파일 오류는 사용자가 다른 파일을 선택하거나 다시 시도할 수 있게 쓴다. `지원하지 않는 파일 형식이에요. 다른 파일을 선택해주세요`처럼 기술 용어를 줄인다.
