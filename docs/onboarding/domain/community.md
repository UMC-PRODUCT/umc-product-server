# Community Domain

## 역할

`community` 도메인은 게시글, 댓글, 번개글, 상장, 신고 기능을 담당한다. 사용자가 활동 내용을 공유하고 커뮤니티 상호작용을 할 수 있게 한다.

## 책임

- 일반 게시글과 번개글을 생성, 수정, 삭제한다.
- 댓글을 작성하고 삭제한다.
- 상장 정보를 관리한다.
- 게시글과 댓글 신고 중복을 막는다.

## 경계

커뮤니티 작성자의 활동 자격은 챌린저 ID로 표현한다. 회원이나 챌린저의 상세 정보가 필요하면 각 도메인의 공개 UseCase를 사용한다.

## UX Writing Notes

커뮤니티는 공개 사용자 표면에 가깝다. 친근한 `~요` 톤을 쓰고, `본인`보다 자연스러운 `내가 작성한 게시글만 수정할 수 있어요`처럼 사용자의 행동을 중심에 둔다.
