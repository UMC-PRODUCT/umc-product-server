# Survey Domain

## 역할

`survey` 도메인은 폼, 섹션, 질문, 선택지, 응답, 답변을 관리한다. 공지 투표에서 분리된 일반 설문/폼 흐름을 담당한다.

## 책임

- 폼을 생성, 수정, 발행한다.
- 질문과 선택지를 관리한다.
- 응답을 임시저장하거나 제출한다.
- 필수 질문, 답변 형식, 응답 가능 기간을 검증한다.

## 경계

survey는 폼 구조와 응답을 소유한다. 파일 첨부 검증은 `storage`, 공지 투표는 `notice` 도메인과 경계를 나눈다.

## UX Writing Notes

폼 작성 문구는 입력 행동을 직접 말한다. `필수 질문에 답변해주세요`, `선택지를 다시 선택해주세요`, `기간을 다시 선택해주세요`처럼 사용자가 수정할 부분을 짚는다.
